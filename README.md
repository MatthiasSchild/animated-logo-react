# Animated Logo in React

This is an example project how you can create an animated SVG logo as a React component.

![Animation](./res/animation.gif)

You can find the component in `src/components`. It consists of a TSX and a SASS file.
The TSX file contains modified the modified code of the logo SVG, which can be found in `res/logo.svg`.

**You can find the tutorial [here](https://matthiasschild.de/animated-logos-as-react-components)!**
